BirbSEO Admin Guide

Admin Dashboard Overview

The BirbSEO Admin Dashboard provides a centralized view of your SEO platform's performance. It allows you to monitor report generation, tool usage, and revenue streams in real-time.

Quick-View Metrics (Top Row)

The top section contains key performance indicators (KPIs) specific to SEO auditing and tool usage.

  • Websites: The total number of external websites currently being tracked or analyzed by your users.
  • Audits: The number of active or recent SEO reports generated in the current month. A high number here indicates high user engagement with your core product.
  • Archived Audits: Historical reports saved in the system (e.g., 78).
  • Tools Views: Measures traffic to your specific free SEO tools (e.g., Keyword Density Checker, SERP Simulator) if you have them enabled publicly.
  • Tools Submissions: Tracks how many times users have actually clicked "Submit" or "Analyze" on your free tools.
  • Users: Total registered account count for the current period.
  • Payments & Earned: Financial metrics displaying the number of transactions and total revenue (USD) collected this month.

Latest Users

A live feed of new registrations.

  • User & Status: Shows the user's identity and whether their account is Active (verified) or requires attention.
  • Plan: Indicates if the user is on a default plan or a Custom tier.
  • Details: Icons representing the user's device, operating system, and geolocation (country flag).

Navigation Sidebar Reference

The sidebar allows you to manage the SEO reports, user subscriptions, and system configurations.

General Management

  • Admin dashboard: The central hub of the panel. Displays the "Mission Control" view with high-level metrics, active user counts, and revenue snapshots.
  • Users: The master list of all registered accounts. Use this to search for members, edit profiles, assign permissions, or suspend access.
  • Settings: Global configuration for the site, including branding, SEO defaults, and system-wide preferences.
  • Plans: Manage subscription tiers. Define limits for "Audits per month" or "Tools usage" for Free vs. Premium users.
  • Languages: Localization settings. Add new languages or translate specific interface strings for international users.

System Tools & Analytics

  • Broadcasts: A mass-communication tool. Use this to send platform-wide announcements or marketing emails to your user base.
  • Users notifications: Configure the specific alerts users receive inside their dashboard (e.g., "Audit Complete" or "Payment Failed").
  • Plugins: The extension manager. Install and configure third-party add-ons to expand functionality without altering core code.
  • Statistics: Visual analytics regarding server performance, visitor geography, and growth trends over time.

Content & Developers

  • Resources: (Dropdown) A built-in content management system. Use this to group content into Categories and create standalone Pages to expand your siteâ€™s informational content.
  • Blog: (Dropdown) A built-in CMS to publish articles, updates, and guides to your public-facing blog pages.
  • API docs: This is the documentation for the available API endpoints, only for the admins, which are built around the REST architecture. All the API endpoints will return a JSON response with the standard HTTP response codes and need a Bearer Authentication via an API Key. Developer API, useful for users looking to integrate your SEO data into their own tools.

Finance

  • Codes: Manage promotional strategies. Create discount coupons for paid plans.
  • Taxes: Configure regional tax rates (VAT/GST) to ensure compliance for payments processed in specific countries.
  • Payments: The financial ledger. View a chronological history of all incoming transactions, download invoices, and all transaction details.

SEO Operations

  • Audits: The master log of all SEO reports generated by the system. Allows admins to review report status or delete specific entries.
  • Archived audits: Access older reports that have been moved to long-term storage to save database resources.
  • Websites: A list of all external domains currently being tracked or analyzed by your users.

Technical & Security

  • Domains: Manage custom domain settings. This controls the "White Label" feature, allowing users to host reports on their own URLs (e.g., audit.agency.com).
  • Notif. handlers: Configure email delivery drivers (SMTP, Mailgun, SendGrid) to ensure system emails avoid the spam folder.
  • Users logs: A security audit trail showing user login times, IP addresses, and device info to help detect suspicious activity.
